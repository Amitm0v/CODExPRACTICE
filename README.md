# DSA-Solutions-Amit-Kumar

A comprehensive collection of Data Structures and Algorithms (DSA) solutions implemented in Python. This repository contains solutions to 42 fundamental and advanced programming problems, ranging from basic number theory to complex linked list manipulations.

## 📌 Project Overview
The goal of this repository is to demonstrate clean, efficient, and understandable Python logic. Every solution is written without internal comments to maintain a "self-explanatory" code structure, focusing on algorithmic thinking rather than built-in shortcuts.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Focus:** Data Structures, Algorithms, Problem Solving

## 📂 Categories Covered

### 1. Number System & Mathematics
* Prime Number verification
* Pythagorean Triplets generation
* Factor calculation and Equation solvers
* Season determination logic

### 2. Array & Matrix Operations
* **Matrix:** 90-degree rotation, Spiral Traversal, and Identity checks.

* **Searching:** Binary Search and Kth Largest element using QuickSelect logic.
* **Optimization:** Maximum Product Subarray, Merge Intervals, and Majority Element (Boyer-Moore).
* **Manipulation:** Moving zeros/hashes, rotating arrays, and finding duplicates.

### 3. String Processing
* **Palindromes:** Basic check and Longest Palindromic Substring logic.

* **Frequency:** Anagram detection, First Unique Character, and String Compression.
* **Structure:** Longest Common Prefix and String Rotation.

### 4. Linked Lists
* Reversing a Singly Linked List.

* Detecting cycles/loops using Floyd’s Cycle-Finding Algorithm.
* Finding the middle node and merging sorted lists.
